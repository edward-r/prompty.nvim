*prompty.nvim*

Prompty integrates the `prompt-maker-cli` binary with NeoVim and
provides streaming buffers for prompt generation and refinement.

COMMANDS                                         *prompty-commands*

:Prompty {intent}            Run the CLI using the given intent. When no
                             intent is provided you will be prompted to
                             enter one.

:PromptyVisual               Capture the current visual selection and
                             use it as the intent text.

:PromptyRefine {text}        Send refinement instructions to the active
                             interactive session. When {text} is omitted
                             the contents of the Prompty instructions
                             buffer are sent instead.

CONFIGURATION                                   *prompty-configuration*

>lua
require("prompty").setup({
  binary = "prompt-maker-cli",
  default_flags = { "--model", "gpt-4" },
  keymaps = {
    prompt = "<leader>pp",
    prompt_visual = "<leader>pP",
    refine = "<leader>pr",
  },
})
<

`binary`           Path or name of the `prompt-maker-cli` executable.
`default_flags`    List of additional flags passed to every CLI run.
`keymaps`          Customize normal- and visual-mode mappings for the
                   provided commands.
